#Beetle-wasp data

From the paper -> https://doi.org/10.1016/j.biocontrol.2019.02.015

Data are from Vansant et al., 2019. They quantified the effect of 6 beetle (host) species on the morphometrics of Dinocampus coccinellae (wasp). Adult beetles were collected from 13 locations in the U.S. 89 individual wasp females and their associated host were weighed and morphometric measurements were taken.

The first column in the data frame is of host species, so it shouldnt be included as input to the NMDS. One another thing to keep in mind, the sample sizes for all the host species were different.

#Square-root Transformation

#Square-root transformation was selected because it gives increased emphasis to less common species, similar to highly abundant species. Wisconsin double-standardization was giving more stress value than square-root transformation alone. 

wasp.sqrt.eu = sqrt(wasp_data[,2:15])

#Performed NMDS analysis on square-root transformed data with wasp weight as the response variable and all other variables as explanatory variables
#Euclidean distance matrix was selected because the data is numerical and continuous
#k indicates the number of dimensions, so in this case they were set to 2. Low-dimensional projections are often better to interpret and are so preferable for interpretation issues.
#Since the data was already transformed, there was no need for another transformation like Wisconsin double-standardization

#NMDS analysis

wasp.nmds.sqrt = metaMDS(wasp.sqrt.eu, distance='euclidean', k = 2, trace = 0,autotransform=FALSE)

#Ordination plot

#NMDS plot looking at wasp weight measures grouped by different Host species
#There are total 89 samples which is why type was kept as 'none' 

wasp_plot = ordiplot(wasp.nmds.sqrt, type='none', 
    main = paste0('Stress = ',round(wasp.nmds.sqrt$stress,dig=3)))

ordiellipse(wasp.nmds.sqrt, kind='ehull', draw='polygon',
    groups=wasp_data$HostSpecies, label=FALSE)

col <- rainbow(length(unique(wasp_data$HostSpecies)))
points(wasp_plot, what='sites', pch=20, col=col[as.numeric(as.factor(wasp_data$HostSpecies))]) 

legend('topright', legend=levels(wasp_data$HostSpecies), pch=20, cex=0.7, col = col)

#In the ordination plot, the space between the groups represent the similarity, so if the groups are ordinated together there is higher similarity. Here, we can see that the wasps from the largest beetles (C. sectumpunctata - in green) among all the beetle species in this study ordinated separately far from the wasps emerging from smallest beetles (H. parenthesis - in pink), showing that the morphometrics of the beetles (host) are dictating the morphometrics of the parasitoid wasps parasitizing them.
